# BombaGit

 A small Git-like version control system written in C.

## Keep in Mind
This is still under construction and not finished !
 ## Installation

 ### Automatic installation

```bash
curl -fsSL https://raw.githubusercontent.com/zehnderdev/BombaGit/main/install.sh | bash
```

You can also clone the repo itself and run the ``install.sh`` script.


## Requirements
- Linux
- GCC
- make
- OpenSSl
